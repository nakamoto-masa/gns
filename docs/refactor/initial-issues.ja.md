# リファクタリングの出発点

2025年11月に実施した設計レビューで判明したコード品質上の問題点

- 設計レビューの調査対象はGNSモデルのみ

## 問題点

- 本来独立しているべき機能の密結合
    - GNSのアルゴリズムの一部がスクリプト上に実装されている
        - 粒子位置の更新
        - KINEMATIC粒子の特別扱い
    - 複数の機能を１つの関数の中にべた書き
        - 設定値の解釈
        - GNSのアルゴリズムの一部
        - トレーニング処理
    - NNモデル本体と、予測モデルとしての外部インターフェース
        - LearnedSimulatorクラス
    - 推論の実行と、推論結果の評価
        - rollout関数
- 型アノテーションの間違い
- グローバル変数による情報の受け渡し
- dict型設定値の多重受け渡し
